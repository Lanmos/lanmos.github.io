求职季3分熟建议

By Lan Chenghao

6月 23 2014 更新日期:6月 23 2014

文章目录

由于公司人事有调动(确切说以前也没有专门人事) 

我临时接手招聘实习生的工作，简历筛了一摞子，也面了不少。

给求职找工作的实习宝宝们叨咕叨咕。

1.简历只有1页或1页多一丢丢的

这种我能打电话叫过来面试 基本上是奔着 简历上 性别:女 (噗嗤开个玩笑谁知道呢……)

正好最近在帮朋友做简历，提供给大家做参考

原:

这种排版是不建议的，要么一页要么就两页，尽量是整篇幅，尾页不要少于三分之二篇幅

简历基本上用表格来布局就ok，但是表格的线框会使得简历看起来很乱，看着就心烦，别说筛选了……

建议大家在网上多筛选一些简历模板，推荐就百度文库里找吧，别找那么花了呼哨，一堆创意的……没用……

改后:

采用的是比较普通的表格排版，隐藏了线框，不同区域采用带底纹的行来标识；

一般分为个人信息、自我评价、求职意向、工作经验、证书、语言能力、兴趣爱好等。顺序可按照不同的行业来组合，像我们专业更重视实习、工作经验，那么经验相关内容会考前，教育经历反倒靠后；自我评价放前放后都可以，但是建议3-4行最优；

经验一定要写出层次，基本上：

①介绍项目或工作的要求

②写自己的职责

③选难点和成就点扩展来写

④一个经验收尾，比如从中掌握到了…加深了…等等 类似大学写实验报告总结那套词；

这部分比重是最大的，如果看一个简历3秒的话，1秒扫基本信息2秒扫经历，如果有照片我会用1分钟来端详……(雄宝宝比我帅的直接甩，呵呵开玩笑的谁知道呢……)



2.卷着简历

就算你不弄个夹子之类的，也别卷着来好吗？虽然你笔试题技术答的不错，通知你复试还是卷着简历来的？

这哥们因为长得不行被pass了……

在这部分插播下：

希望女生穿的颜色靠一点，别那么花如果是行政请自带丝袜

男的来了请暂停抠鼻挖耳屎，有礼貌一点

服饰得体就好，正装、工装懂？不一定是西装……没钱的话休闲也没有问题啦

3.抖腿的滚粗

我上班没事也抖一抖但是不管你平时啥样，你面试克制一点

文职类：如果是做行政、人事、会计等的还是看看xxx期的公务员面试视频，学个样子就ok;

技术类：比如我们程序员，有个坐样大方点别拘谨就ok。

4.关于态度

面试嘛，你可能就是海投过来的，也不见得我们就一定彼此相中是吧，就像处对象，是奔着结婚去的，但中间处着不适合了也不能硬凑不是；但你的态度一定要是积极的，就算你是装的，最好也表现的积极一点，诚恳一点，比如我们遇到过这样来面试的：

“我本来打算去天安门的，看地图离这近就打算过来了”——你当我屌丝的时间就不宝贵是吧？

“我回去考虑一下，明天还有一个面试，等它出结果了再告诉你好吧”——我擦，啥意思没进公司就玩起跳槽预告来了？

5.大方一点

就是说话别太小家子气，怎么解释呢，就是提高点音量，最好是面带微笑，身体靠前坐一点

遇到过这样的:雄宝宝，说话声音有点柔和，可以理解，双手垂下或抠着拨了盖，额……我一看不知道为什么突然有了尿意……

你最好能在面试的时候多说，面试的时间是固定的，你说的多，我问的不就少了吗！复试时间不固定，但是你说的多，还是好一些，废话别太多！对提问的细节伸展即可，让面试官跟你有沟通了解的欲望；

6.对于经历、能力的提问

前面我们说到了工作经验很重要，但是刚毕业在大学啥也没干的你，除了翘课吃包子有经验，貌似也没啥可写的，但是你编也得编不编也得编！别管真假有了”经历“你得好好分析下你编的那些经历，多在网上找些参考，起码跟你的”经历经验“混个脸熟嘴溜。

说实话我们一般很少细细的研究写的那个，但是对里面的一些疑难点，面试的时候可能会重点考察，以我们的职位为例子，你在工作经历上写了开发过一个权限系统，那好我冷不丁的看见了我突然我就问了：这个权限系统你负责哪个部分？涉及几个数据表？表之间怎样的关系，第一层你回答不错，接着就是第二层，层次越来越深，如果考官有一定水平的话，可能会问具体的实现细节。

如果我想考察你的学习能力，我一般先问：你觉得你的学习能力怎么样？

百分百的童鞋回答到：我觉得我学习能力挺强的。（你丫学习能力这么强至于来我们这小单位吗……）

我接着会问：那你说说java基础部分由哪几部分组成，或者它基础部分的知识体系是怎么样的（主要是为了间接考察下他学东西的思路）

如果我想考察你的工作思路，前几天有个网页设计的来，我说如果我让你设计一个类似中粮网的页面，把你的页面配色方案跟我说说

答：嗯……绿的，我会在网上多找找类似的参考……

大哥这个在网上找参考也是一种办法，但是不是作为你解决问题的主要思路；如果在方案确定会议上，你找了3 - 4个参考，很难说服大家来确定某一个，最终方案有可能流产，你的工作能力也会受到怀疑；

我给他的方案是，中粮网这种类似食品类商城的界面，你可以选一种蔬菜、水果；在ps里用吸管来抽取不同部分的颜色，做色签，由于出自一个事物，这些抽取出来的颜色会很自然，搭配起来不会很突兀；这样你在会上解释起来也有说辞，你能引导大家来确定；对于思路本身来说，什么样的思路都可以，它的意义在于让你接到任务不至于无所适从，它能让你阶段性的、可操作性的靠近最终要的目标；

7.你的规划

我问10个人，未来3-5年有神马职业规划吗？

10个人里面有11个回答：我要当项目经理……

尼玛，我还想当董事长呢！

建议宝宝们在找工作之前，对你所从事的职业先做个调查，这个职业有哪些职位，分别是什么，分别都有哪些上升路线可走，就是所谓的T线；然后你把3-5年跟上升的路线做个匹配就OK了，举个例子，拿我们搞软件的来说:

①先百度 程序员发展路线 得到技术路线，实施路线，销售路线，产品路线，运维路线等，你选几条就行

②每个路线的阶段是什么或者说目标职位是什么：

技术路线：初、中、高级程序员，系统分析员，系统架构师，技术经理，技术总监

产品路线：产品设计师(界面设计、流程设计、系统设计、体验设计)，产品监理，产品经理，产品总监

管理路线：初、中、高级程序员，系统分析员、项目经理助理、项目经历、项目总监

（阶段和目标以你们调研综合的为主，不同阶段的职位名称也可能不同）

③分析下每个阶段要达到这个阶段职位目标所需要的一些能力，比如初级程序员需要编程基础、中级程序员需要面向对象设计思维、高级程序员需要熟练应用设计模式(面很广，但是你得摘出来几个)

④把3-5年往上套吧

最后就是：我计划3年内，第一年加强基础编程的能力，以升为中级程序员为目标;第二年以升为高级程序员为目标，并提高自己对公司业务的掌握程度;第三年以产品经理助理为目标，分析市面优秀应用设计，结合本公司业务不断磨合锻炼自己的产品设计能力，希望能够履行自己以产品经理为目标的职业规划；

8.如果你啥也不会

真的，我遇到一个干了半年人事，觉得人事没发展，又想投身开发的一个姐们，问题是她只准备了一天……

你要是啥也不会，可真挺头疼……如果你有一个月准备时间，那么一定要好好利用这一个月复习基础，起码能白话出来，不然……我也不知道你的命运如何，当然有可能比啥都会的还牛逼，不过那是后话了……

宝宝们可以面试2周或者3周后再正式开始，前面可以投一些小公司或者一些不感兴趣的简历，海投一下，有机会面试就当个锻炼，主要锻炼你的衣着、态度、说话等等；如果是工、理科，笔试就算打不上也得多写点思路！！交白卷可就没机会翻身了！

9.如果你觉得面试过了，要问我们一些什么？

待遇一定不要忘了！但是，在这之前你先卖个萌，问个什么咱们公司的团队氛围和企业文化啦，这样让人觉得，嗯这个小伙不那么现实

然后就是问到待遇，一定要问清楚实习、试用、签合同的时间！实习是3个月还是拿到毕业证就算完事(法律规定是拿到毕业证，但是在tz你懂得……)，问清楚福利，三险一金还是五险一金，什么时候交，交多少！迟到扣不扣钱！请病假扣不扣？加班有钱吗？加班没钱的话有调休吗？有餐补吗？谁带你？等等……

别因为着急找工作，而把自己卖给黑心户啊~~

基本上就这些了，希望能帮助宝宝们求职!
